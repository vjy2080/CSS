Border in CSS

  • The border shorthand CSS property sets an element's border.
  • It sets the values of border-width, border-style, and border-color.
  • We can use it saperately as following :-
    1. border-width
    2. border-style
    3. border-color

Outline in CSS

  • Outlines never take up space, as they are drawn outside of an element's content.
  • According to the spec, outlines don't have to be rectangular, although they usually are.
  • The outline shorthand CSS property sets an element's outline.
  • It sets the values of outline-width, outline-style, and outline-color.
  • We can use it saperately as following :-
    1. outline-width
    2. outline-style
    3. outline-color
