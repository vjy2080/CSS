Column

  • The columns CSS shorthand property sets the number of columns to use when drawing an element's contents, as well as those columns' widths.

    1. Column Width

      The ideal column width, defined as a < length > or the keyword auto. The actual width may be wider or narrower to fit the available space.

    2. Column Count

      The ideal number of columns into which the element's content should be flowed, defined as an < integer > or the keyword auto. If neither this value nor the column's width are auto, it merely indicates the maximum allowable number of columns.
