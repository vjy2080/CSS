The CSS background properties are used to define the background effects for elements. There are lots of properties to design the background.

CSS background properties are as follows:

  1. CSS Background-color Property:

    2. The background-color property in CSS is used to specify the background color of an element.

  2. CSS Background-image Property:

    3. The background-image property is used to set one or more background images to an element.

  3. CSS Background-repeat Property:

    4. The background-repeat property in CSS is used to repeat the background image both horizontally and vertically.

  4. CSS Background-attachment Property:

    5. The background-attachment property in CSS is used to specify the kind of attachment of the background image with respect to its container.

  5. CSS Background-position Property:

    6. In CSS body-position property is mainly used to set an image at a certain position.

  6. CSS Background-origin Property:

    7. The background-origin is a property defined in CSS which helps in adjusting the background image of the webpage.

  7. CSS Background-clip Property:

    8. The background-clip property in CSS is used to define how to extend background (color or image) within an element.
