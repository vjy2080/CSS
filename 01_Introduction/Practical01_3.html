Example of External CSS

  1. External CSS (thick red border) given in external stylesheet for <span> tag

    2. <link rel="stylesheet" href="style.css" class="style">

    <p>Lorem ipsum <span>dolor</span> sit amet.</p>

    Lorem ipsum dolor sit amet.

  2. External CSS (Yellow Background with larger font) given in external stylesheet for <span> tag

    3. <link rel="stylesheet" href="style.css" class="style">

    <p>Lorem ipsum dolor sit amet consectetur, <span class="spanclass">adipisicing elit </span>. Dolore, commodi.</p>

    Lorem ipsum dolor sit amet consectetur, adipisicing elit . Dolore, commodi.

  3. External CSS (Big font & Center alignment) given in between style tag in head for <h5> tag

    4. <link rel="stylesheet" href="style.css" class="style">

    <h5>Lorem ipsum dolor sit amet.</h5>
    Lorem ipsum dolor sit amet.
