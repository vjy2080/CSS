Example of Inline CSS

  1. Inline CSS (red font color) given to <span> tag

    <p>Lorem ipsum <span style="color: red;">dolor</span> sit amet.</p>

    Lorem ipsum dolor sit amet.

  2. Inline CSS (black border) given to <span> tag

    <p>Lorem ipsum dolor sit amet consectetur, <span style="border: 2px solid black;">adipisicing elit </span>. Dolore, commodi.</p>

    Lorem ipsum dolor sit amet consectetur, adipisicing elit . Dolore, commodi.

  3. Inline CSS (Blue Background & Yellow font) given to <h5> tag

    <h5 style="background-color: blue; color: yellow">Lorem ipsum dolor sit amet.</h5>
    Lorem ipsum dolor sit amet.
