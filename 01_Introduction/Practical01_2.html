Example of Internal CSS

  1. Internal CSS (red font color) given in between style tag in head to <span> tag

    2. <head><style>span{color: red;}</style></head>

    <p>Lorem ipsum <span>dolor</span> sit amet.</p>

    Lorem ipsum dolor sit amet.

  2. Internal CSS (black border) given in between style tag in head to <span> tag

    3. <head><style>.spanclass{border: 2px solid black;}</style></head>

    <p>Lorem ipsum dolor sit amet consectetur, <span class="spanclass">adipisicing elit </span>. Dolore, commodi.</p>

    Lorem ipsum dolor sit amet consectetur, adipisicing elit . Dolore, commodi.

  3. Internal CSS (Blue Background & Yellow font) given in between style tag in head to <span> tag

    4. <head><style>h5{background-color: blue;color: yellow;}</style></head>

    <h5>Lorem ipsum dolor sit amet.</h5>
    Lorem ipsum dolor sit amet.
