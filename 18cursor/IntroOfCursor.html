What is cursor in CSS? and it's uses

  • The cursor CSS property sets the mouse cursor,
  • If any, to show when the mouse pointer is over an element.
  • The cursor setting should inform users of the mouse operations that can be performed at the current location,
  • Including: text selection, activating help or context menus, copying content, resizing tables, and so on.
  • You can specify either the type of cursor using a keyword,
  • Or load a specific icon to use (with optional fallback images and mandatory keyword as a final fallback).

Formal syntax of Cursor

  • cursor: url(name.jpg) x-axis y-axis, auto;
  • cursor: url(name.jpg) x-axis y-axis, default;
